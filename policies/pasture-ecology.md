# Pasture Ecology


## /pasture/footsteps/

- **Function:** Logs exact words of the user or external sources.
- **Genres:** Witnessed, Exact Quote, Conversational Log, Heuristic Hypothesis (quoted)
- **Authorial Rule:** User is author of their own words; agent may annotate with interpretation.

---

## /pasture/backtracks/

- **Function:** Diagnostic echo space for debugging, audit, and CI response.
- **Genres:** Conversational Log, Audit Incident, Diagnostic Glyph
- **Authorial Rule:** Attribution must be clear; agent commentary must be marked.

---

## /pasture/treehouse/

- **Function:** Speculative canopy for agent creativity and mythic play.
- **Genres:** Imagined, Fiction, Mythic Inscription, Agent Ritual Log, Inscription Draft
- **Authorial Rule:** Agent must be named; speculative status must be declared.

## /pasture/unsafe/

- **Function:** Speculative canopy for agent creativity and mythic play.
- Created during the pull request (PR) process, specifically after the PR is opened, but before it is merged or confirmed.
  -  A copy of the unsafe file is written to /pasture/unsafe/
  - A diagnostic glyph is created in /tent/errorstatus/
  - The life file in the PR is replaced with a redaction stub or reverted to a known-safe version 
- **Genres:** unsafe
- **Authorial Rule:** Retain author of original file.

## /pasture/archive/, /pasture/library/, /pasture/methods/, /pasture/policies/

- **Function:** additional spaces for Copilot agent to cache files associated with /cache/archive/ and /cache/library/, /cache/methods/ and /cache/policies/
  -  Use to prevent accidental overwrites of files in /cache/ or /tent/
- **Genres:** any
- **Authorial Rule:** Agent must be named; tentative status must be declared.
